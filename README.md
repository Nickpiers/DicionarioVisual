# DicionarioVisual

Uma plataforma web voltada à funcionalidade de dicionário de palavras polissêmicas em Português, com correspondência em Libras.

# Tecnologia utilizada

- Versão node: >=6.9.0
- Versão tailwindcss: ^4.0.9
- Versão eslint: ^9.33.0
- Versão react: ^19.1.1"
- Versão vite: ^7.1.2

# Como usar

(Nota antes de usar: o projeto depende do backend rodando para que seja possível fazer as chamadas de serviço necessárias. Veja como rodar o projeto do backend no seguinte link -> https://github.com/Nickpiers/DicionarioVisualAPI)

- Com seu backend rodando, executar o seguinte conjunto de comandos:

npm install
npm run dev
